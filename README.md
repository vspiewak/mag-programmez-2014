mag-programmez-2014
===================

Analyse des logs applicatifs avec ELK - Dossier DevOps - Programmez 2014

Vous devez installer les programmes suivant:
* Virtual Box
* Vagrant

Lancez la VM de démo:
    
    vagrant up


Stoppez la VM de démo:

    vagrant halt


Supprimez la VM de démo:

    vagrant destroy
